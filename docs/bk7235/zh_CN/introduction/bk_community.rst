Github 社区
-----------------------

在 Armino Github Issue Forum: https://github.com/bekencorp/armino 上，您可以：

 - 报告 Armino 相关问题
 - 提交一个新特性开发请求
 - 提交一个改进建议

技术论坛
-----------------------

博通集成技术论坛 <http://bbs.bekencorp.com/forum.php> 旨在提供一个开放式的技术交流平台，您可以在这里您可以
发表与 Armino 技术相关的建议，观点，或者寻求技术支持。
