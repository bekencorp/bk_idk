# ARMINO 物联网开发框架

* [English Version](./README.md)

欢迎使用 Armino!
Armino 是**博通集成**推出的物联网开发框架，旨在支持**博通集成** 2022 年之后推出的各种物联网芯片，
同时兼容各种主流物联网操作系统，如 RT-Thread，AliOS，HarmoneyOS 等，Armino 默认支持 FreeRTOS。
bk_idk为Armino生态的基础SDK。
